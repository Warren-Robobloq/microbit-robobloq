# robobloq

这是控制Robobloq的micro:bit积木
 
日期: 2018.7

## 使用方法
1.打开 makecode 编辑器，在项目中选择添加软件包，然后在地址栏输入下面网址  
https://makecode.microbit.org/
2.然后添加包：
3.输入：
https://github.com/moklib/microbit-robobloq
回车确认

搜索后就可以添加并使用本软件包了。
